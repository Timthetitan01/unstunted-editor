UNSTUNTED EDITOR
NLE Video Editor

Lightweight, offline Premiere-style editor.
AI captions · Lumetri Color · WebCodecs export · No subscription.

⬇  Download for Windows
v0.1.0  ·  ~140 MB  ·  NSIS installer
⚠️ ARM64 only — runs on Snapdragon X / Surface Pro X / Windows on ARM.
Standard Intel / AMD PCs need the x64 build (coming soon).

⬇  Download for macOS
v0.1.0  ·  ~160 MB  ·  DMG disk image
⚠️ Apple Silicon only — M1, M2, M3, M4 and later.
Intel Mac builds coming soon.

curl -L https://github.com/Timthetitan01/unstunted-editor/releases/latest/download/Unstunted.Editor-0.1.0-arm64.dmg -o UnstuntedEditor.dmg && hdiutil attach UnstuntedEditor.dmg && cp -R /Volumes/Unstunted\ Editor/Unstunted\ Editor.app /Applications/ && hdiutil detach /Volumes/Unstunted\ Editor

Minimum

OS — Windows Windows 10 ARM64 (21H1+)
OS — macOS macOS 11 Big Sur (Apple Silicon)
CPU ARM64 — Snapdragon 8cx Gen 3 / M1
RAM 4 GB
GPU Any DirectX 11 / Metal capable GPU
Storage 2 GB free (app + Whisper model)
Display 1440 × 900
Internet Required once — to download Whisper AI model (~142 MB). Fully offline after.

Recommended

OS — Windows Windows 11 ARM
OS — macOS macOS 14 Sonoma
CPU Snapdragon X Elite / M2 or newer
RAM 16 GB
GPU Hardware H.264 encoder (VideoToolbox / Media Foundation)
Storage 20 GB+ free for media projects
Display 1920 × 1080 or higher
Internet Not required after first run
Note on AI transcription (Whisper) The first time you use captions, the app downloads the Whisper base model (~142 MB) and caches it to disk. After that it runs 100% offline. Transcription speed scales with CPU — M2 / Snapdragon X Elite will be roughly 2–3× faster than minimum-spec.
Export performance Export uses WebCodecs hardware H.264 encoding when available (fast) and falls back to FFmpeg software encoding. Hardware encoding requires a supported GPU (VideoToolbox on Mac, Media Foundation on Windows ARM). Software fallback works on all machines but is slower.
