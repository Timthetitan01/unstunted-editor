Recommended

OS — Windows Windows 11 ARM
OS — macOS macOS 14 Sonoma
CPU Apple M2+ / Snapdragon X Elite
RAM 16 GB
GPU H.264 hardware encoder
Storage 20 GB+ for media
Display 1920 × 1080 or higher
Internet Not needed after first run
AI Transcription (Whisper) Downloads the Whisper base model (~142 MB) the first time you use captions, then caches it to disk. Fully offline after that. Transcription is 2–3× faster on M2 / Snapdragon X Elite vs. minimum spec.
Export Uses WebCodecs hardware H.264 encoding (fast) when available, with automatic FFmpeg software fallback on any machine.
